# 秋秋很开心 · 重连文模板

## 适用场景

断更半年以上后的第一篇回归文章。

## 标题公式

不是"好久不见，这半年我去哪儿了"（被动、道歉感）
→ **"好久不见，我旅居中国一年了！"**（主动、有姿态、时间跨度更大）

规则：动词 + 时间段 + 感叹号。例：`好久不见，我{做了什么}{多久}了！`

## 结构

```
大家好，我是秋秋。

打开后台一看，上次更新已经是半年前了。
（开门见山，承认 gap，一行就够，不道歉）

1、在{做某件大事}
（具体地点 + 精准数字 + 实拍图 + 时间线）
- 去年五月 → 北京骑行威海 | 800多公里10天 | 配骑行图
- 威海住了半年 → 两室一厅1500月租 | 26度不用空调 | 配海景图
- 去年十月买车 → 5个箱子全部行李 | 配行李图
- 一脚油门往南开 → 5000多公里开了一个月 | 每城玩几天
- 海南住了三个月 → 三亚五星酒店淡季500一晚 | 月花销2万
- 海南→云南大理 → 租别墅 | 推门满眼绿色 | 配大理图
- 收束："所以过去一年，我一直在过喜欢的生活。"

2、在{另一件在做的事}
（关联其他平台/账号 → 自然过渡到新方向）
- "小红书账号「XX」一直有更新"（核实平台！可能是小红书非公众号）
- 写的内容：旅居开销/利息覆盖生活费/理财心得
- 回到本号，有时间就继续分享
- 引出新方向："现在AI时代盛行，我们需要学习操作的更多了。"
- "如何利用AI接管自己的生活、有哪些省时间的小工具。"
- "可以分享的内容太多了！"
- 品牌口号：给新方向取一个名字（如"AI接管者"）

接下来的我......
（心态转变叙述 → 具体钩子 → 不是课程大纲）
- 以前："我发现了一个好东西分享给你"（旧模式）
- 现在：技术变成了生活的一部分（新模式）
- 引用一个外部权威："XX就说过，未来十年..."（第三方背书，可选但加分）
- 如何利用AI工具 = 普通人拉开差距的地方
- 每天写东西，AI帮我查资料理思路
- 读了一本好书，AI自动整理成笔记
- 听播客看文章也不用自己动手整理了
- 三个具体钩子（跟后面文章方向一致）：
  1. 怎么让读过的书、播客不再过目就忘？
  2. 怎么让零散日常事务不操心？
  3. 怎么两小时从想法到上线小工具？
- 总结表格（可选）：`| 总结 | 1. 没更新的日子也在过喜欢的生活 2. 未来风口是AI接管者 |`
- 结尾：✨ 转发给你关心的人 / 喜欢记得星标
- 签名：秋秋很开心
```

## 关键规则

| # | 规则 | 错误示例 | 正确示例 |
|---|------|---------|---------|
| 1 | **数字打底** — 每段叙事后跟真实数字 | "三亚酒店很便宜" | "三亚五星酒店淡季500一晚，月花销差不多两万" |
| 2 | **不要推任何产品** — 这是重连，不是销售 | 提复利计算器、小程序 | 只说"两小时从想法到上线"（抽象不点名） |
| 3 | **不要道歉过多** — 承认 gap 一行就够 | "对不起大家我消失半年真的对不起" | "上次更新已经是半年前了。" |
| 4 | **不要课程大纲** — 用生活节奏代替分类目录 | "方向还是三个：工具方法、效率、成长" | "1、在环游中国" "2、在记录日常" |
| 5 | **AI 直接正面说** — 不回避不躲藏 | "大老王搭的系统" "技术手段" | "AI帮我查资料" "AI接管自己的生活" |
| 6 | **大老王至多提一次** — 在无关紧要的场景一笔带过 | 每段都提大老王 | 只在骑行那段提一次"和大老王骑行过去" |
| 7 | **给新方向命名** — 让读者记住一个概念 | "我会分享AI相关" | "AI接管者" |
| 8 | **配图密集** — 每处地点都配图，比想象中更多 | 6张图 | 14+张：骑行/海景/行李/路线/酒店/大理各一 |
| 9 | **引用名人背书** — 加分项，可选 | 整篇自己说 | "孙宇晨就说过，未来十年..." |
| 10 | **核实其他平台** — 小红书 vs 公众号，不说错 | 猜"另一个公众号" | 先确认再写"小红书账号" |
