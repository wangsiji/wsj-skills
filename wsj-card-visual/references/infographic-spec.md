# 信息长图生成规范（Step1-3）

用户提供的知识流程图信息长图一键生成三步骤。用于风格 C（深色信息长图风）的内容结构化流程。

## Step1：输入文本结构化拆解

接收用户输入的系统/教程文字，自动拆分5类固定模块：

1. **标题区**：主标题+副标题+宣传slogan
2. **理论总述区**：核心底层逻辑/闭环原理
3. **分阶段流程区**：按步骤拆分1/2/3/4阶段，每阶段配小标题+分项要点
4. **案例/库示例区**：名词解释+使用案例+结构样例
5. **底部总结区**：核心价值、闭环流程图、适用人群

**拆解规则**：
- 每个模块控制3-6行要点，每行短句，拒绝大段文字
- 区分「小标题（加粗）」「正文说明」「功能标签」三类文字层级

## Step2：标准化排版文案（适配长图版式）

**固定版式**（和题图完全对齐）：
- 顶部大标题：粗体大号字，主标题紫蓝渐变，小字灰色副标题
- 顶部标识栏：左侧工具图标+加号+右侧成果图标
- 分栏布局：左右两栏/上下多区块，每块带彩色圆角标题栏（紫/蓝/绿/橙区分阶段）
- 区块内部结构：
  1. 模块标题栏（彩色底色+白色加粗文字）
  2. 小标题（加粗黑色）
  3. 圆点列表分项说明（短句，一行一个信息点）
  4. 配套极简图标：文件夹、代码框、时钟、大脑、文档
- 底部三栏：核心价值、闭环流程图、适用人群
- 最底部页脚：浅色小字宣传标语
- 色彩规范：深色背景（深灰/黑色底色），文字白色，分区标题使用低饱和马卡龙紫/蓝/绿/橙，图标浅白简约线性风

## Step3：AI绘图提示词

```
信息科普长图，深色炭黑色背景，模块化圆角卡片排版，分5大彩色分区，
每个分区顶部有彩色标题栏，搭配简约线性扁平化图标；
内容是AI知识库搭建教程图文，分阶段文字说明，文字清晰可阅读，无模糊扭曲字体；
扁平矢量插画风格，极简商务UI，干净柔和低饱和配色，
紫色、蓝色、薄荷绿、浅橙色区分不同模块；
布局从上至下：大标题区、理论区、4个流程阶段区块、底部总结三栏；
卡片圆角，细微内阴影，精致科技感，知识教程海报，高清，图文均衡，文字分层清晰，长竖版9:16构图
禁止：文字扭曲、字体模糊、杂乱元素、写实照片、3D厚重模型
```
