### 自动检查清单（每次日报必过）

1. **80/20 检查** — Z2(E区) TL占比 < 80% 则警告。测试/比赛周特殊处理：高强度日占比可超过20%是合理例外，但高强度后必须≥48h恢复，且测试次日只做恢复跑（非轻松跑），TL控制在30以内。
2. **连续天数** — ≥4天强制建议休息
3. **Z3灰色区漂移** — 轻松跑/恢复跑标记为灰色区(Z3) → 警告跑太快
4. **里程碑** — 剩余≤14天标⏰并给出倒计时安排
5. **高强度间隔** — 检查48h恢复规则是否满足
6. **COROS HR区标签验证** — 比赛/测试跨步跑等高强度活动，即使COROS基于平均心率标记为"E区"，也要根据配速判断真实强度。全力10km测试（配速≈阈配速4'25"）平均心率偏低是正常现象（前1-2km心率爬升慢拉低均值），应判定为T区强度，不被"E区"标签误导。
7. **训练频率检查** — 过去7天训练≤3次且连续休息≥2天 → 提示频率偏低，建议逐步增加到每周4-5次。休息日可安排3-5km恢复跑维持连续性，比完全休息好。
8. **减重赤字检查** — 参考 `03-Resources/Notes/运动周追踪表.md` / `02-Areas/12-饮食.md`：用户目标 158→125斤。若用户口述体重，记录趋势；若连续2周无下降，提示检查隐形热量（坚果/水果/奶茶）或跑量不足。减重是破三最高杠杆（1kg≈3秒/km），优先级与跑步量并列。
9. **肩带力量检查** — 用户目标改善溜肩（弹力带+单杠，无哑铃）。每周应≥2次肩带力量（肩袖外旋/侧平举/面拉/悬垂/引体），单次≤30min，放在跑步后或休息日，不造成中枢疲劳影响长跑。若周报显示肩带训练为0且跑步≥5次，提示插入。
